# Compositional AI Systems with Verifiable Safety Properties
## Research Programme — Modulith Research CIC

### Mission
Develop the mathematical framework proving that modular compositional AI architectures
provide verifiable safety guarantees that monolithic systems cannot.

### Scope
This repository contains:
- Mathematical specifications of the modular system under study
- Formal proof agenda and proof sketches
- Sanitised experimental data for proof validation
- Paper drafts for publication
- Verification tooling

### What This Repo Does NOT Contain
- Implementation code (proprietary, separate repository)
- Training procedures
- Model weights or checkpoints
- System architecture details beyond mathematical abstraction

### Structure
```
specs/           — Mathematical specification documents
proofs/          — Formal proofs (LaTeX)
data/            — Sanitised experimental data for validation
papers/          — Paper drafts
tools/           — Verification scripts (black-box API only)
literature/      — Relevant papers and references
```
