# Sanitised Experimental Data

Data files in this directory are provided for proof validation.
All data is pre-processed to remove implementation details.

## Available Datasets
- pool_expansion.csv — Loss values as pool grows from 7 to 47 modules
- leave_one_out.csv — Per-domain ablation results
- memory_discrimination.csv — Memory module routing weights (own vs unrelated)
- routing_entropy.csv — Policy entropy over training

Data is generated by the PI from the proprietary system.
Request additional data via the PI if needed for proofs.
